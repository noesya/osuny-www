Osuny permet de réaliser et de gérer des sites web, c’est donc un système de gestion de contenu (ou CMS pour Content Management System) comme il en existe d’autres, WordPress étant le plus connu. Mais certains aspects de la solution ont été particulièrement soignés et expliquent les résultats obtenus, notamment la sobriété, la sécurité, l’accessibilité et la qualité de l’expérience. 

notes: >- alt: >- credit: >- - template: chapter title: >- Accessibilité position: 2 data: layout: no_background text: >-

Un site produit avec Osuny est très accessible, avec 96% des critères RGAA respectés pour le site de l’IUT Bordeaux Montaigne en janvier 2023. Le respect des bonnes pratiques techniques est intégré dans le thème Osuny, et nous améliorons en continu la qualité grâce aux contributions des personnes du réseau. Les contributions sont accompagnées dans l’outil d’administration, avec un système de vérification de l’accessibilité qui indique les défauts lors des contributions (transcriptions manquantes pour les vidéos, balises alt possiblement incorrectes...).


notes: >- alt: >- credit: >- - template: chapter title: >- Sobriété position: 3 data: layout: no_background text: >-

Les sites sont statiques, produits avec Hugo, ce qui diminue l’impact de l’hébergement. Le thème est extrêmement léger, avec moins de 20 requêtes (dont une partie est mise en cache, ce qui diminue encore l’impact après la première page) et moins de 500 ko par page (en mobile). Pour parvenir à ce résultat, les images sont particulièrement optimisées. Enfin, les sites fonctionnent sur des périphériques très anciens, ce qui est une façon pragmatique de lutter contre l’obsolescence.

notes: >- alt: >- credit: >- - template: chapter title: >- Sécurité position: 4 data: layout: no_background text: >-

Du fait de l’architecture d’Osuny, les sites Web sont extrêmement sécurisés : il n’y a pas de langage serveur ni de base de données, et aucune information confidentielle. Uniquement un paquet de fichiers HTML, CSS et JS hébergés sur une instance serveur. C’est à la fois facile à héberger pour les établissements et très rassurant.


notes: >- alt: >- credit: >- - template: chapter title: >- Design position: 5 data: layout: no_background text: >-

Les sites utilisent un système de blocs de contenus, qui sont riches et responsives. Les interfaces intégrées par défaut dans le thème Osuny sont partagées en Creative Commons dans un document Figma. Sur la base de cette interface, des designers peuvent adapter et modifier librement les fonctionnements, de façon à obtenir un système de design spécifique à l’établissement, respectant parfaitement les choix typographiques, coloriels et iconographiques. Dans le cas d’un établissement qui opère plusieurs sites Web, il est possible de créer un thème “enfant” du thème Osuny, qui reprend toutes les bonnes pratiques, ajoute la couche visuelle spécifique et peut ensuite être réutilisé simplement pour chaque site.
