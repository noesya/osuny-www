Chez noesya, nous pensons qu’un numérique d’intérêt général est possible. Il faut pour cela lui donner sa juste place, ni plus ni moins. Cela passe par un travail exigeant sur le code, le design et la gouvernance du web.

notes: >- alt: >- credit: >- - template: chapter title: >- Exigence position: 2 data: layout: no_background text: >-

C’est pourquoi nous travaillons tous nos projets avec l’exigence :

  • de décarboner massivement le web,
  • de rendre les sites accessibles à tous les publics
  • d’améliorer la sécurité face aux risques cyber
  • d’ouvrir le code
  • de soigner l’expérience utilisateur
  • de construire des communs numériques 
notes: >- alt: >- credit: >- - template: chapter title: >- Osuny position: 3 data: layout: no_background text: >-

Nous fabriquons Osuny, une solution technique spécialement conçue pour les universités, laboratoires de recherches et écoles supérieures pour leur permettre de créer des sites Web entièrement personnalisés, les plus sobres, les plus accessibles et les plus sécurisés possibles.
