Osuny, comme WordPress, K-Sup, ou Ametys, est une solution Open Source. Le code d'Osuny est libre et gratuit, vous pouvez l'utiliser comme vous le souhaitez dans le cadre de la licence MIT. 

notes: >- alt: >- credit: >- - template: chapter title: >- Payer les prestations de développement ad hoc position: 2 data: layout: no_background text: >-

Vous pouvez solliciter n'importe quel prestataire pour produire un site appuyé sur Osuny, cela n'a aucun impact sur le prix de développement. Chaque prestataire devise son travail en fonction de ses propres choix, tant pour le développement initial que pour la maintenance du site (TMA). 

notes: >- alt: >- credit: >- - template: chapter title: >- Bénéficier de l'hébergement souverain en SaaS position: 3 data: layout: no_background text: >-

noesya propose un hébergement souverain d'Osuny avec Scalingo et Scaleway, sans limite de nombre de personnes utilisatrices, comprenant la maintenance du back-office, la publication vers Github ou Gitlab, le CDN et l'hébergement Netlify en front (si nécessaire). Cette mutualisation des coûts est proposée selon la grille suivante :

  • Université (sites illimités) 7 500€
  • Groupe privé (sites illimités) 15 000€
  • Ecole publique (1 site) 1 500€
  • Ecole privée (1 site) 3 000€
  • Association (1 site) 500€
