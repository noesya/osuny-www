Nous sommes très heureux de vous annoncer la mise en ligne du nouveau site de l'IUT Bordeaux Montaigne !

alt: >- credit: >- buttons: - title: >- Découvrir le site url: >- https://www.iut.u-bordeaux-montaigne.fr target_blank: true - template: chapter title: >- Sécurisé position: 3 data: text: >-

Le site est précompilé, c'est à dire que c'est un ensemble de fichiers HTML, sans langage serveur ni base de données. Il n'y a pas de vulnérabilité de la couche applicative, parce qu'il n'y a pas de couche applicative : pas de Java, pas de PHP, pas de MySQL, PostgreSQL...

L'outil d'administration est hébergé sur le data center Parisien d'Outscale, certifié ISO 27001. Les couches systèmes sont administrées par Scalingo. L'application est dotée d'authentification multi-facteurs. Le code est surveillé par Code Climate pour la qualité et par Github pour la sécurité.
