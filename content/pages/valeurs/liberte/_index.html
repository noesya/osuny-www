--- title: > Liberté url: "/valeurs/liberte/" position: 5 description: > Ouvrir le code --- Osuny est une plateforme Open Source, en licence MIT. Les technologies utilisées pour construire Osuny sont elles aussi en source ouverte, à l'exception du thème graphique utilisé pour l'admin (Appstack), pour lequel noesya a acquis une licence d'utilisation compatible avec un projet Open Source. Le back-office est construit avec Ruby, Ruby on Rails, PostgreSQL, et un ensemble de gems consultables sur le référentiel Github du projet. Le front-office est construit avec Hugo, Bootstrap, et le moins possible de dépendances, listées sur le référentiel Github du thème.
